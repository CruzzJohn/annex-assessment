# Annex Technologies Assessment (JavaScript Version)

This repository contains JavaScript implementations of the Annex Technologies technical assessment.

## Contents

### 1. pth_factor.js
Finds the p-th smallest factor of a number using O(√n) optimization.

### 2. top_students.js
Returns top 3 students sorted by score (desc) and ID (asc).

### 3. top_articles.js
Fetches and ranks articles from HackerRank API based on:
- number of comments (desc)
- article name (desc)

## Notes
- Code is structured for readability and interview discussion
- Includes edge case handling and basic error safety
- Designed as a prototype before conversion to C/C++
